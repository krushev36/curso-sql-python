# 📋 Programa del Curso: SQL para Ciencia de Datos usando Databricks

**Institución:** Maestría en Ciencia de Datos e Inteligencia de Negocios  
**Duración:** 14–16 horas (8 sesiones de ~2 horas)  
**Modalidad:** Práctica — 100% en Databricks Free Edition  
**Prerrequisitos:** Conocimientos básicos de programación; NO se requiere experiencia en SQL ni bases de datos

---

## 📌 Descripción General

Este curso introduce SQL como herramienta fundamental para el análisis de datos en la era del Big Data. El estudiante aprenderá a consultar, transformar y analizar grandes volúmenes de datos utilizando Databricks, la plataforma líder en análisis de datos a escala.

A diferencia de un curso tradicional de bases de datos, este programa está orientado **exclusivamente al análisis y ciencia de datos**, con un enfoque práctico y basado en casos de negocio reales.
